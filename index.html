Introducing Pico.

A stupidly simple & blazing fast, flat file CMS. Making the web easy.

Stupidly Simple

Picos makes creating and maintaining a website as simple as editing text files.

Blazing Fast

Picos is seriously lightweight and doesn't use a database, making it super fast.

No Database

Picos is a "flat file" CMS, meaning no database woe's. no MYSQL queries, nothing.

Markdown Formating

Edit your website in your favorite text editor using simple markdown formating.

Twig Template

Pico usees the twig templating engine. for powerful and flexible themes.

Open Source

Pico is completely free and open source, relased under the MIT license.
